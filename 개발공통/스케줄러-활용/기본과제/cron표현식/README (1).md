### 스케줄러 활용

cron 표현식 정리

```
● * : 모든 값을 뜻함.
● ? : 특정한 값이 없음을 뜻함.
● - : 범위를 뜻합니다. (예) 월요일~수요일 : MON-WED로 표현
● , : 특별한 값일 때만 동작 (예) 월,수,금 : MON,WED,FRI 로 표현
● / : 시작시간 / 단위  (예) 0분부터 매 5분 : 0/5
● L : 일에서 사용하면 마지막 일, 요일에서는 마지막 요일(토요일) (Last 개념)
● W : 가장 가까운 평일 (예) 15W : 15일에서 가장 가까운 평일을 찾음.
● # : 몇째 주의 무슨 요일을 표현 (예) 3#2 : 2번째주 수요일
```

```
 ┌────────── minute (0 - 59)
 │ ┌──────── hour (0 - 23)
 │ │ ┌────── day of month (1 - 31)
 │ │ │ ┌──── month (1 - 12)
 │ │ │ │ ┌── day of week (0 - 6 => Sunday - Saturday, or
 │ │ │ │ │                1 - 7 => Monday - Sunday)
 ↓ ↓ ↓ ↓ ↓
 * * * * * /경로 설정(실행할 명령)
```
